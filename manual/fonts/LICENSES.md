# Font Licences — konCePCja Manual

This manual ships **only freely redistributable fonts** (Strategy A in the build
plan). The sibling CPC6128 manual's commercial faces — Rockwell N90 (Monotype),
Century Schoolbook BT (Bitstream), Madison Antiqua Pro (Linotype), OCR-B Pro
(Monotype) — are **deliberately not included**; free substitutes are used instead.

| File | Role | Font | Licence | Source |
|------|------|------|---------|--------|
| `Rokkitt[wght].ttf` | Headings (slab-serif, substitutes Rockwell) | Rokkitt | SIL OFL 1.1 | Google Fonts `ofl/rokkitt` |
| `texgyreschola-*.otf` | Body (substitutes Century Schoolbook BT) | TeX Gyre Schola | GUST Font License (LPPL-like, free) | CTAN `fonts/tex-gyre` |
| `OCRB.otf` | Code / listings | OCR-B | Matthew Skala (2011, 2021), after Norbert Schwarz (1986, 2011) — free* | tsukurimashou.osdn.jp / sibling manual |
| `JuliaMono-Regular.ttf` | Keycap glyphs | JuliaMono | SIL OFL 1.1 | github.com/cormullion/juliamono |
| `CascadiaCode-Regular.ttf` | Control-code pictures | Cascadia Code | SIL OFL 1.1 | github.com/microsoft/cascadia-code |
